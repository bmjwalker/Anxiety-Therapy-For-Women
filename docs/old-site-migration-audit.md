# Old Site Migration Audit

This checklist maps the legacy site to the current brand site before the old domain is retired.

## Current mapping status

- Home page (/): Covers the main homepage narrative, services, programs, about, resources, FAQ, and blog content.
- About (/about): Added as a dedicated page so the legacy About URL continues to work.
- Resources (/resources): Added as a dedicated page so the legacy Resources URL continues to work.
- FAQ (/faqs): Added as a dedicated page so the legacy FAQ URL continues to work.
- Blog (/blog): The main blog listing and legacy archive are already present.
- Program pages: Existing program pages cover the key paid offerings from the old site (Reset Circle, Intensive, Group, etc.).

## What still needs manual review

1. Verify the old contact/consult booking links still point to the live intake flow.
2. Confirm the old Good Faith Estimate and Disclaimer links remain current and publicly accessible.
3. Re-check the old blog archive and program pricing language against the live intake page before launch.
4. Add any last missing testimonials, credentials, or resource links if the old site has additional content not yet surfaced in the new brand site.

## Recommendation

Keep the old site live until the new route pages above are validated in production, then redirect legacy URLs to the new brand site pages.
